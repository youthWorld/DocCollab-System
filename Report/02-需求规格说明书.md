# 需求规格说明书（SRS）

## 一、引言

### 1.1 目的

本文档旨在详细描述"多人文档协作系统（DocCollab）"的功能需求与非功能需求，为后续的设计、开发、测试提供依据。本文档遵循CMMI需求管理过程域的要求，确保需求的完整性、可追溯性和可验证性。

### 1.2 范围

DocCollab 是一款面向研究生课程小组项目的轻量级协作管理工具，以"版本容器"为核心概念，提供文档协作编辑、文件管理、版本追溯等功能。

### 1.3 定义与缩写

| 术语     | 说明                                                    |
| -------- | ------------------------------------------------------- |
| 版本容器 | 保存一次完整项目状态的快照单元，包含文档、代码、资源等  |
| SRS      | Software Requirements Specification，软件需求规格说明书 |
| BC       | BroadcastChannel，浏览器原生跨标签页通信 API            |

---

## 二、总体描述

### 2.1 产品视角

本系统为独立的Web应用程序，采用前端浏览器本地存储（LocalStorage），无需后端服务器。通过 BroadcastChannel API 实现同浏览器多标签页实时协作，用户通过浏览器直接访问。

### 2.2 用户特征

- **主要用户**：研究生课程小组成员
- **技术水平**：具备基本的计算机操作能力
- **使用场景**：课程项目协作，包括文档编写、代码管理、资源整理

---

## 三、功能需求

### 3.1 v1.0 核心功能（必须有）

#### R1：版本容器管理

| 属性         | 内容                                                                                                                                             |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **编号**     | R1                                                                                                                                               |
| **功能名称** | 版本容器管理                                                                                                                                     |
| **版本**     | v1.0                                                                                                                                             |
| **优先级**   | 高                                                                                                                                               |
| **描述**     | 进入项目时系统自动生成 v1.0 初始工作区容器。用户可主动"保存新版本"，系统将当前工作区状态打包为新版本容器。每个容器包含完整的文件列表和文档内容。 |
| **输入**     | 容器名称                                                                                                                                         |
| **处理**     | 系统创建新的版本容器，将当前工作区的所有内容打包保存为新版本                                                                                     |
| **输出**     | 新的版本容器，包含版本号、创建时间、包含的文件列表                                                                                               |
| **约束**     | 容器名称不可为空                                                                                                                                 |

#### R2：文档协作编辑

| 属性         | 内容                                                                                                                                    |
| ------------ | --------------------------------------------------------------------------------------------------------------------------------------- |
| **编号**     | R2                                                                                                                                      |
| **功能名称** | 文档协作编辑                                                                                                                            |
| **版本**     | v1.0                                                                                                                                    |
| **优先级**   | 高                                                                                                                                      |
| **描述**     | 基于 TipTap 富文本编辑器，支持加粗/斜体/下划线/标题/列表/引用/代码块/撤销重做。采用 BroadcastChannel API 实现同浏览器多标签页实时同步。 |
| **输入**     | 文档内容编辑操作                                                                                                                        |
| **处理**     | 通过富文本编辑器进行文档编辑，变更实时同步到其他协作用户                                                                                |
| **输出**     | 更新后的文档内容                                                                                                                        |
| **约束**     | 基于浏览器LocalStorage存储，协作限于同一浏览器环境                                                                                      |

#### R3：资源上传下载

| 属性         | 内容                                                                                  |
| ------------ | ------------------------------------------------------------------------------------- |
| **编号**     | R3                                                                                    |
| **功能名称** | 资源上传下载                                                                          |
| **版本**     | v1.0                                                                                  |
| **优先级**   | 高                                                                                    |
| **描述**     | 支持代码文件、图片、视频等资源文件的上传（转 base64）和下载，存储在 LocalStorage 中。 |
| **输入**     | 本地文件                                                                              |
| **处理**     | 将文件读取转 base64 并存储到 LocalStorage，关联到当前容器                             |
| **输出**     | 上传成功的文件列表；下载时输出原始文件                                                |
| **约束**     | 受浏览器本地存储容量限制（通常数百MB），单个大文件建议不超过50MB                      |

---

#### R4：版本标签与说明

| 属性         | 内容                                                                                                                        |
| ------------ | --------------------------------------------------------------------------------------------------------------------------- |
| **编号**     | R4                                                                                                                          |
| **功能名称** | 版本标签与说明                                                                                                              |
| **版本**     | v1.0                                                                                                                        |
| **优先级**   | 中                                                                                                                          |
| **描述**     | 每次"保存新版本"必须选择版本标签（初始版本/功能迭代/Bug修复/文档完善/代码重构）并填写版本说明，确保每个版本都有明确的目的。 |
| **输入**     | 版本标签、版本说明文本                                                                                                      |
| **处理**     | 系统校验标签和说明非空，将标签和说明关联到新版本容器                                                                        |
| **输出**     | 带有标签和说明的版本记录                                                                                                    |
| **约束**     | 标签为必选项，说明为必填项（不少于2个字符）                                                                                 |

#### R5：变更记录

| 属性         | 内容                                                                                                                                                         |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **编号**     | R5                                                                                                                                                           |
| **功能名称** | 变更记录查看                                                                                                                                                 |
| **版本**     | v1.0                                                                                                                                                         |
| **优先级**   | 中                                                                                                                                                           |
| **描述**     | 以看板卡片形式展示所有版本，点击展开可查看：版本说明、本版本变更明细（相比上一版本的新增/删除/修改的文件和文档）、文件快照、文档快照。支持一键切换到该版本。 |
| **输入**     | 选择查看的项目                                                                                                                                               |
| **处理**     | 系统从存储中读取该项目所有版本容器，计算相邻版本差异明细                                                                                                     |
| **输出**     | 看板列表 + 每版本变更明细                                                                                                                                    |
| **约束**     | 无                                                                                                                                                           |

#### R6：版本对比

| 属性         | 内容                                                                                                         |
| ------------ | ------------------------------------------------------------------------------------------------------------ |
| **编号**     | R6                                                                                                           |
| **功能名称** | 版本对比                                                                                                     |
| **版本**     | v1.0                                                                                                         |
| **优先级**   | 中                                                                                                           |
| **描述**     | 选择任意两个版本进行对比，系统展示差异：新增文件、删除文件、新增文档、删除文档、修改文档（含新旧内容对比）。 |
| **输入**     | 选择两个版本号                                                                                               |
| **处理**     | 系统对比两个版本的文件列表和文档内容，计算差异                                                               |
| **输出**     | 差异对比结果（新增/删除/修改清单）                                                                           |
| **约束**     | 只能对比同一项目内的两个版本                                                                                 |

| 属性         | 内容                                                                                                                              |

## 四、非功能需求

### 4.1 性能需求

| 编号 | 需求         | 指标                    |
| ---- | ------------ | ----------------------- |
| NFR1 | 响应速度     | 页面加载时间不超过3秒   |
| NFR2 | 文档编辑响应 | 输入延迟不超过100ms     |
| NFR3 | 存储容量     | 支持至少200MB总存储空间 |

### 4.2 可用性需求

| 编号 | 需求       | 指标                                   |
| ---- | ---------- | -------------------------------------- |
| NFR4 | 易用性     | 新用户无需培训可在10分钟内完成基本操作 |
| NFR5 | 界面一致性 | 所有页面使用统一的UI组件库             |

### 4.3 数据安全

| 编号 | 需求       | 说明                                   |
| ---- | ---------- | -------------------------------------- |
| NFR6 | 数据本地化 | 所有数据存储在浏览器本地，不上传服务器 |
| NFR7 | 数据导出   | 支持将数据导出为JSON文件备份           |
